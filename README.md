# Infoviz system based on open-source technologies
## How to run?
1. Clone repository
2. Install composer dependencies:
```php
    composer install
```
3. Run migrations:
```php
    php artisan migrate
```
4. Install NPM dependencies:
```php
    npm install
```
5. Build frontend:
```php
    npm run production
```
